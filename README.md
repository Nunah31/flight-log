# יומן טיסות — אודי הברי

## העלאה ל-Netlify (5 דקות, חינמי)

### שלב 1 — GitHub
1. כנסי ל-github.com והירשמי (חינמי)
2. לחצי על "+" → "New repository"
3. שם: `flight-log` → לחצי "Create repository"
4. לחצי "uploading an existing file"
5. גרירת **כל הקבצים** מהתיקייה הזו לדף → "Commit changes"

### שלב 2 — Netlify
1. כנסי ל-netlify.com → "Sign up with GitHub"
2. לחצי "Add new site" → "Import an existing project"
3. בחרי GitHub → בחרי את `flight-log`
4. Build command: `npm install && npm run build`
5. Publish directory: `dist`
6. לחצי "Deploy site"
7. אחרי ~2 דקות מקבלים קישור! 🎉

### שלב 3 — התקנה על הנייד (אנדרואיד)
1. פתחי Chrome בנייד
2. היכנסי לקישור שקיבלת מ-Netlify
3. תפריט (3 נקודות) → "הוסף למסך הבית"
4. לחצי "הוסף" — מופיע אייקון על המסך!
5. שלחי את אותו קישור לאודי לעשות אותו דבר
